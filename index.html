O que são juros?
Os juros representam o preço do dinheiro ao longo do tempo. Essa é a taxa que incide no valor inicial emprestado para uma pessoa ou empresa. Nessa relação, existem duas partes principais: o credor, que é o responsável por disponibilizar o dinheiro, e o tomador, que solicita o empréstimo. Para oferecer os recursos para o tomador, o credor cobra uma taxa que faz o dinheiro render, a taxa de juros. Para entender melhor, considere que uma instituição financeira disponibiliza R$ 1.000 para um cliente na forma de empréstimo pessoal. Em troca, ele precisa pagar uma taxa de 10% ao mês. Se a pessoa demorar 30 dias para fazer o pagamento do empréstimo, ela pagará R$ 1.100 para o banco. O lucro da instituição será esse montante de R$ 100, que foi obtido a partir dos juros. Esse é um exemplo simples, que desconsidera os diferentes tipos de cálculo, mas que serve para demonstrar como os juros agem.
